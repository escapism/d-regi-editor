## Dレジエディター
同人イベント用レジアプリ「[Dレジ](https://github.com/escapism/d-register)」の頒布物情報を編集するためのWebエディターアプリです。  
一からデータを作成するほか、「Dレジ」で書き出したJSONを読み込んで、表計算ライクな画面で編集することができます。  
また、同一ブラウザ内であれば「Dレジ」から直接頒布物情報をインポートすることもできます（ただしiPhone、iPadでは制約があります）。

[GitHub Pages で公開しています](https://escapism.github.io/d-regi-editor/)

編集後は再びJSONを書き出し、「Dレジ」でインポートしてください。

CSVの書き出し・読み込みにも対応しているので、エクセルなどの表計算ソフトを使っての編集も可能です。

**※アプリ内でのデータ保存機能はありません。**

[Dレジはこちら](https://github.com/escapism/d-register)

### ショートカットキー
- Win [Ctr + ; (+)] / Mac [⌘ + ; (+)] 行を追加。フォーカスがある場合はその下に追加。
- Win [Ctr + -] / Mac [⌘ + -] フォーカスがある行を削除。
- Win [Ctr ↑→↓←] / Mac [⌘ ↑→↓←] セル移動

## 推奨環境
スマホでも使用できますが、基本的にはPC、またはタブレットでの操作を想定しています。  
PCの場合はChromeの使用をおすすめします。
